## Features
* **Responsive Design:** Fully optimized layout that adjusts seamlessly for mobile, tablet, and desktop screens using CSS Flexbox.
* **Project Showcase:** Dynamic section highlighting key projects like the Automated Price Tracker.
* **Clean UI/UX:** Minimalist design with smooth navigation and professional typography using Google Fonts.
* **Contact Integration:** Direct links to LinkedIn, GitHub, and Email for easy recruitment outreach.
